# Juan Cardona Portfolio

This is the personal portfolio website of Juan Cardona, showcasing professional projects, skills, and contact information. The portfolio is built using HTML, CSS, and Bootstrap 5 for a clean, responsive, and modern design.

---

## Features

- Responsive navigation bar  
- Portfolio section to showcase projects  
- Contact form for visitors to send messages  
- Social media links for professional networking  
- Mobile-friendly and clean UI  

---

## Technologies

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons
